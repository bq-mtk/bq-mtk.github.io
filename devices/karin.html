Aquaris M8 (karin)

2021-04-05 :: Jose
#specs  #Android  #Ubuntu 

The BQ Aquaris M8 (codenamed "karin") is a mid range tablet from BQ. It was released in November 2016.

Basic Spec Sheet
CPU 1,3Ghz Quad-Core ARM Cortex-A53
CHIPSET Mediatek MT8163B 1,3 GHz
GPU MediaTek ARM Mali-T720 MP2 520MHz
Kernel 3.18.22
RAM Memory 2GB
Shipped Android Version 6.0
ROM Memory (Storage) 16GB
SDCard Support Yes
Battery LiPo 7280 mAh (non-removable)
Display 800 x 1280 pixels, 8" IPS
Rear Camera 5.0 MP
Front Camera 2.0 MP
