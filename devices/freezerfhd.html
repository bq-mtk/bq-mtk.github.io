Aquaris M10FHD (freezerfhd)

2021-04-05 :: Jose
#specs  #Android  #Ubuntu 

The BQ Aquaris M10FHD (codenamed "freezerfhd") is a mid range tablet from BQ. It was released in November 2015.

Basic Spec Sheet
CPU 1,5Ghz Quad-Core ARM Cortex-A53
CHIPSET Mediatek MT8163A 1,5 GHz
GPU MediaTek ARM Mali-T720 MP2 600MHz
Kernel 3.18.22
RAM Memory 2GB
Shipped Android Version 5.1, Upgradable to 6.0.1
ROM Memory (Storage) 16GB, 32GB
SDCard Support Yes
Battery LiPo 7280 mAh (non-removable)
Display 1200 x 1920 pixels, 10" FHD
Rear Camera 8.0 MP
Front Camera 5.0 MP
