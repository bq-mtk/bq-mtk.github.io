Aquaris E5 (vegetahd)

The BQ Aquaris E5 (codenamed "vegetahd") is a low-mid range smartphone from BQ. It was released in July 2014.

Basic Spec Sheet
CPU 1,3Ghz Quad-Core ARM Cortex-A7
CHIPSET Mediatek MT6582 1,3 GHz
GPU MediaTek ARM Mali-400 MP2 500Mhz
Kernel 3.10.108
RAM Memory 1GB, 2GB
Shipped Android Version 4.4, Updatable to 5.1
ROM Memory (Storage) 8GB, 16GB
SDCard Support Yes
Dual SIM Support Yes
Battery LiPo 2500 mAh (non-removable)
Display 720 x 1080 pixels, 5" HD
Rear Camera 16.0 MP
Front Camera 8.0 MP
